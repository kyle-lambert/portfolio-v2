Web development

A modern and user-friendly website to browse your favourite movies, television shows and actors.

React.js, Redux, SCSS, Framer Motion, TMDB API

Prior to the development of this project, I was still fairly new the concepts of React and how the pieces of a front-end framework fit together. With these thoughts in mind, I decided it would be a great idea to build a movie search website and thus MovieCloud was born. I figured MovieCloud would provide value to the community as well as present some tough challenges along the way. As I look back now I can safely say I've learnt many things...
